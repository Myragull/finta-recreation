# Finta Landing Page

A modern, responsive landing page built with **Next.js**, **Tailwind CSS**, and **TypeScript**.  
This project demonstrates a clean, professional UI layout with reusable components, optimized images, and interactive elements for a SaaS-style product landing page.

## Description

This landing page includes:

- A responsive **Navbar** with dynamic links and a call-to-action button
- A **Hero section** with heading, description, and interactive buttons
- Reusable **Container component** for consistent layout across sections
- Optimized images using `next/image` with lazy loading and draggable disabled
- Conditional Tailwind class management using `clsx` and `tailwind-merge`
- Fully responsive design for desktop and mobile screens

## Features

- Reusable **Navbar** and **Hero** components
- Dynamic link mapping using arrays and `.map()` method
- Tailwind CSS for styling with hover effects, shadows, rounded corners, and transitions
- Optimized images for performance (`next/image`)  
- Conditional classes using `clsx` + `tailwind-merge`
- Responsive and mobile-friendly layout
- Easy to extend with new sections or components


## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- clsx
- tailwind-merge
